Positioning:

Watch out! All positioning except "fixed" are relative to the first positioned enclosing container.

Static:

The default positioning in the browser. Block elements get staked up vertically, while inline elements horizontally.

Relative:

Positioning relative the the normal(static) position in the page. The space taken by the element is preserverd.

Absolute:

It doesn't take into account other elements. The space of the positioned element is not preserved.

Fixed:

See at the upper rigth corner. It remains on the page when scrolling. The space of the positioned element is not preserved.
